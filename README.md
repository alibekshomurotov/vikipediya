# 🔬 Mashhur Olimlar Ensiklopediyasi

## 📌 Sayt haqida

**Mashhur Olimlar Ensiklopediyasi** - dunyoning eng buyuk olimlari haqida to'liq va ishonchli ma'lumot beruvchi zamonaviy veb-ilova. Ushbu sayt orqali siz:

- ✅ 20 dan ortiq mashhur olimlar haqida ma'lumot olishingiz
- ✅ Olimlarning tarjimai holi, kashfiyotlari va mukofotlari bilan tanishishingiz
- ✅ O'zbek va Ingliz tillarida ma'lumot o'qishingiz
- ✅ Dark/Light mode da qulay ko'rinishda ishlashingiz
- ✅ Qidiruv orqali kerakli olimni tez topishingiz mumkin

**Ma'lumotlar manbai:** Wikipedia API - ishonchli va doimiy yangilanib turadigan ma'lumotlar bazasi.
